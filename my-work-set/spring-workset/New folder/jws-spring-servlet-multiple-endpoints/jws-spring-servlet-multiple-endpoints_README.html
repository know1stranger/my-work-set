Project-Desc

This project is using jws and spring for supporting multiple endpoints. Setup is sample for exporting servlet based service(webservice) in non JEE 5 Env.  (According to reading doc this setup it will work in tomcat server as it will have the JWS runtime.)

Project-Impl

The webservice is spring maintained bean exposed using WSSpringServlet (RI from JAX-WS). Webservice is delegating call to other spring managed bean pojo service.
Unlike traditional config this setup will only need one servlet in the web.xml to support multiple enpoint urls
