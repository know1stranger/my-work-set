Project-Desc

This is simple project using jws and spring. Setup is sample for exporting standalone service(webservice)

Project-Impl

The webservice is spring maintained bean exposed using spring without servlet at the back. Running the main client(MyStandaloneWSServer.java) will allow to access the webservice.
The webservice itself is spring maintained bean and exposed using its default settings localhost,baseport (for instance). The webservice is delegating call to spring managed bean. (@Webservice is given to the spring SEI( i.e.)service endpoint interface.)
@Service annotation is used by the spring to load the service(s).
@Webserice annotation is used to configure for the spring impl. In this case as we are exposing the service endpoints with the spring inbuilt SimpleJaxWsServiceExporter. It will register endpoint with the internal JAX-WS runtime server and publishes for us.
